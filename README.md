# Battery Inventory Management System

## Overview

The Battery Inventory Management System is a Python and MySQL based application developed to manage battery inventory records efficiently. The system allows users to store, update, delete, and retrieve battery-related information through database integration.

## Technologies Used

* Python
* MySQL

## Features

* Add battery inventory records
* Update existing records
* Delete records from inventory
* View and manage inventory data
* Database integration using MySQL

## Project Structure

* `Batteryinventory.py` - Main Python application
* `Batteryinvertor.sql` - Database schema and SQL queries
